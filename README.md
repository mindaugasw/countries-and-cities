# Šalys ir miestai

Paleistas projektas: [salys-ir-miestai.herokuapp.com](https://salys-ir-miestai.herokuapp.com/)

### Instaliavimas
Reikalingas XAMPP stack.  
1. `git clone git@github.com:mindaugasw/SalysIrMiestai.git`  
2. Nustatyti DB prisijungimo duomenis (`config-default.php`).  
3. Importuoti DB (`DB_export.sql`).  


### Duomenys
Šalių ir miestų sąrašas gautas iš [simplemaps.com/data/world-cities](https://simplemaps.com/data/world-cities). Visa kita informacija (šalių ir miestų plotai, gyv. skaičius, tel. kodai, pašto kodai) sugeneruoti atsitiktinai.
